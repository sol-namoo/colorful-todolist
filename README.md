# Coloful Todo-list
할 일을 daily, study, work 카테고리로 구분해서 관리할 수 있도록 만들었습니다.<br/>
각 카테고리에 작성한 할 일와 체크 여부는 로컬스토리지에 저장됩니다.

- 편리한 UI로 Todo를 쉽게 생성하고 삭제할 수 있습니다.<br/>
- 카테고리별로 Todo를 설정할 수 있습니다.<br/>
- create-react-app으로 간편한 번들링과 배포가 가능합니다.<br/>
